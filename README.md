# Applied Data Science Journey | DataCamp

## Overview

This repository documents my applied learning journey in Data Science through hands-on projects, exercises, and real-world case studies completed on DataCamp.

The objective of this repository is not to showcase course content, but rather the practical application of concepts learned in Data Analysis, Data Visualization, Machine Learning, Data Engineering, and MLOps.

Each project focuses on solving a specific business or analytical problem while following industry-oriented workflows, from data preparation and exploratory analysis to modeling, evaluation, and communication of results.
